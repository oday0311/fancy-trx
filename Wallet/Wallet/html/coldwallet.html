How does offline signing work?

Ready for work:

  1. Two phones without root or you think it’s safe.
  2. Installed the newest version and download from google play or App store.
  3. One of phone open airplane mode as cold wallet,keep it offline always because it's saved your privacy key. Watch wallet

If you want to use cold wallet in RayboWallet version ,you should watch the wallet at first.

note:one of phone connected network as hot wallet to watch cold wallet,the other one keep offline for Authorized signature.

Step1 Hot wallet:click “import wallet” -> choose watch wallet ->click “scan button” on upper-right corner ,scan wallet address QR code from step2

Step2 Cold wallet:choose the wallet you want to be watched

Step3 Hot wallet:scan step2’s cold wallet->click“import wallet”->

Cold wallet transaction signature

Let’s use offline signature function to transfer TRX:

Step1 Hot wallet:choose one watch wallet that is the same with cold wallet->go to TRX transaction page,input all the transfer information,click “next” button->generate QR code

Step2 Cold wallet:scan step1’s QR code from hot wallet->click”sign”->input password->generate “successful signature” QR code

Step3 Hot wallet: click”next”button->click”scan”button,then scan “successful signature” QR code from cold wallet generate in step2->click “confirm transaction” button

TIPS:

  1. There is saved privacy key in cold wallet phone, so keep it safe and complete backup carefully
  2. Keep cold wallet offline
  3. Watch wallet can’t finish transaction without “offline signature” from cold wallet
